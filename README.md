# 🥗 NutriGuide – Smart Nutrition Assistant

## 📌 Overview

**NutriGuide** is a web-based application designed to help users make healthier food choices by providing nutritional insights, meal planning suggestions, and personalized tracking features. It simplifies nutrition management through an easy-to-use interface and efficient backend processing.

---

## 🚀 Features

* 🔐 User Registration & Login System
* 📊 Personalized Dashboard
* 🧮 Nutrition Calculator (Calories, Protein, Carbs, etc.)
* 🍽️ Meal Planning Suggestions
* 📷 Food Scanner (Concept-Based Feature)
* 💾 Secure Data Storage using Database

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS

**Backend:**

* Python (Flask Framework)

**Database:**

* MySQL

## ⚙️ Installation & Setup

1. **Clone the Repository**

```bash
git clone https://github.com/rishwanththota96-oss
cd nutriguide
```

2. **Create Virtual Environment (Optional but Recommended)**

```bash
python -m venv venv
venv\Scripts\activate   # For Windows
```

3. **Install Required Packages**

```bash
pip install flask mysql-connector-python
```

4. **Setup Database**

* Create a MySQL database (e.g., `nutriguide`)
* Create required tables (users, nutrition data, etc.)
* Update database credentials in your Flask app

5. **Run the Application**

```bash
python app.py
```

6. Open browser and go to:

```
http://127.0.0.1:5000/
```
